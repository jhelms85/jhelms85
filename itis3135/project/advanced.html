Import Statement

The import statement in Python is used to include external modules or packages into your program. Modules are Python files containing functions, classes, and variables that can be used in other Python files.

By importing modules, you can access their functionality and reuse code, saving time and effort in development.

Let's explore how the import statement works:

Example 1: Importing the entire module

import math print(math.sqrt(16)) # Output: 4.0

Example 2: Importing specific functions from a module

from random import randint print(randint(1, 10)) # Output: Random integer between 1 and 10

Example 3: Giving a module an alias

import datetime as dt print(dt.datetime.now()) # Output: Current date and time

Mean, Median, Mode, and Range

Python provides built-in functions to calculate statistical measures like mean, median, mode, and range.

  • Mean: The average value of a dataset.
  • Median: The middle value of a dataset when arranged in ascending order.
  • Mode: The value that appears most frequently in a dataset.
  • Range: The difference between the maximum and minimum values in a dataset.

Let's see how to calculate these:

Mean

numbers = [2, 3, 5, 7, 11, 13] mean = sum(numbers) / len(numbers) print("Mean:", mean)

Median

sorted_numbers = sorted(numbers) n = len(sorted_numbers) median = sorted_numbers[n // 2] if n % 2 != 0 else (sorted_numbers[n // 2 - 1] + sorted_numbers[n // 2]) / 2 print("Median:", median)

Mode

from collections import Counter mode = Counter(numbers).most_common(1)[0][0] print("Mode:", mode)

Range

range_val = max(numbers) - min(numbers) print("Range:", range_val)

Randomization

Python's random module allows you to work with random numbers and perform various randomization tasks.

Some common randomization tasks include generating random integers, shuffling elements, and selecting random samples.

Let's look at some examples of randomization:

Generating a Random Integer

import random random_number = random.randint(1, 100) print("Random number:", random_number)

Shuffling a List

my_list = [1, 2, 3, 4, 5] random.shuffle(my_list) print("Shuffled list:", my_list)

Selecting a Random Element from a List

random_element = random.choice(my_list) print("Random element:", random_element)
