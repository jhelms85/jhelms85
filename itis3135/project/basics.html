Python Basics

Data Types

Now that we've looked into some syntax, the next thing we will dive into is some basics. The first thing we will cover here will be Python data types.

There are numerous built in data types, let's take a look at them.

Basic Data Types
Type: Syntax:
Text str
Number int, float, complex
Boolean bool
Sequence list, tuple, range
Less Common Data Types
Type: Syntax:
Binary bytes, bytearray, memoryview
Set set, frozenset
None NoneType
Mapping dict

Again, lets break it down.

Text

str

type used to represent sequences of characters, typically text, enclosed within single quotes ('..') or double quotes ("..").

Number

int

type used to represent integers (whole numbers without decimals).

float

type used to represent floating-point numbers (numbers with fractional parts).

complex

type used to represent complex numbers (numbers with a real and imaginary parts).

Boolean

bool

type used to represent boolean values, which can either be True or False.

Sequence

list

type used to represent ordered, mutable collections of items. Defined with square brackets [..].

tuple

type used to represent ordered, immutable collections of items. Defined with parentheses (..).

range

type used to represent an immutable sequence of numbers. Generated using the built-in range() function.

Note that in Python, when we say a data type is "mutable," it means that its contents can be changed after it's created. "Immutable" means that the data type's contents cannot be changed after creation. Lists are mutable, meaning you can add, remove, or modify elements. Tuples, on the other hand, are immutable, meaning once created, their elements cannot be changed.

Binary

bytes

type used to represent an immutable sequence of bytes. Used to represent binary data. Defined with a prefix b'...'.

bytearray

type used to represent a mutable sequence of bytes. Similar to bytes but can be modified after creation.

memoryview

type used to represent a memoryview objects which allow Python code to access the internal data of an object that supports the buffer protocol without copying.

Set

set

type used to represent an unordered collection of unique elements. Defined with curly braces {...} or by using the set() constructor.

frozenset

type used to represent an immutable set. Similar to set but cannot be modified after creation.

None

NoneType

type used to represent the absence of a value, often used as the return type of functions that don't explicitly return anything.

Mapping

dict

type used to represent a collection of key-value pairs. Also known as a dictionary. Defined with curly braces {key: value} or by using the dict() constructor.

We will get a little more into these shortly, and see some examples of each of them. But first lets figure out how to set these data types. In Python data types are set when you assign values to variables, in example when you type this:

x = 5

The data type of 'int' is automatically assigned by Python.

Now what about this?

x = "John"

The data type of 'str' will be assigned in this case, since a str value was used.

Functions

In Python, functions are a block of reusable code that performs a specific task. They allow you to break down your program into smaller, manageable parts, making your code more organized and easier to understand.

Definition of Functions:

A function in Python is defined using the def keyword followed by the function name and parentheses. Any parameters should be placed within the parentheses. The code block that follows the function definition is the body of the function.

Syntax of Defining a Function:

def function_name(parameters):
                Function body: Code block that performs the task
                return [expression]

Parameters and Arguments:

Parameters are placeholders in the function definition. Arguments are the actual values passed to the function when it is called.

Return Statement:

The return statement is used to exit a function and return a value (if any) to the caller. If the return statement is not present, the function will return None by default.

Calling a Function:

To call a function, simply use the function name followed by parentheses and any required arguments.

Example:

def add_numbers(a, b):
            """This function adds two numbers."""
            return a + b
        
        result = add_numbers(3, 5)
        print(result)  # Output: 8

Basic Math Calculations

Next, we will get into some basic built in Python math functions, addition, subtraction, division, multiplication, remainder.

Python can perform many calculations for you! This is important because calculations will also be used in many other important functions you may use, such a looping! We will cover this a little later on in the "Advanced" section of this course.

Let's go through some quick examples!

Addition and Subtraction

If you were to type:

print(5 + 2)

You would get this returned:

7

Pretty simple right? Well let's see how variables would work in this instance. For the input:

x = 5
print(x + 2)

You would get this returned:

7

So as you can see, by declaring the variable, you can do the same calculations. By declaring them this way though, you can easily use the same variables for multiple calculations, this is especially useful when you are dealing with much larger numbers.

As for subtraction, well it works the exact same way.

print(8 - 2)

You would get this returned:

6

That is really all there is to it with addition and subtraction, with Python it is extremely easy to perform these calculations.

Multiplication, Division, and Remainder

Now, let's explore some more basic arithmetic operations in Python: multiplication, division, and finding the remainder.

These operations are fundamental in programming and can be used in various scenarios to perform calculations.

Let's dive into some examples!

Multiplication

To perform multiplication in Python, use the * operator:

print(3 * 4)

This will output:

12

Similarly, you can use variables for multiplication:

y = 3
print(y * 4)

Which will also result in:

12

Division

Division in Python can be performed using the / operator:

print(10 / 2)

Which yields:

5.0

It's important to note that division in Python results in a floating-point number.

Remainder

To find the remainder of a division operation, you can use the modulus operator %:

print(10 % 3)

This gives:

1

The remainder operator is particularly useful in tasks like checking for divisibility or cycling through values.
