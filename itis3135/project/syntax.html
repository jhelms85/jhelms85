What does this look like?

As we saw on the overview page, Python has a relatively simple syntax when compared to other programming languages. Because of this, it often is the first language to be taught. Here is the example we looked at:

def hello_world():
    print("Hello, World!")

We already broke this down previously, but here is a refresher in case you need it.

defthis is a keyword in python used to define a function.

hello_worldthis text here is used to define the function's name.

( ) :The parenthesis (empty in this example) are used to set parameters for the function, and the colon is to define where the function's executable code begins.

printthis is a built in Python function that is used to display information or content to the screen.

("Hello, World!")Hello, World! would be the exact text that is displayed to the screen on execution of the function.

Example:

Correct:

if 5 > 2:
    print("TRUE!")

Incorrect:

if 5 > 2:
    print("TRUE!")

In the above example, the first statement is correct as it uses indentation properly before the print statement. The amount of spaces can differ and as the programmer you can decide, the common amount of spaces is four, and most programmers use this as the standard. In most IDE's pressing tab will be set to four spaces, and many auto indent for you based on context. So in other words, after you type your first line:

if 5 > 2:

and press the Enter key,

The IDE will drop down a line and add an indent and you can begin typing the print statement.

if 5 > 2:

    print("True!")
