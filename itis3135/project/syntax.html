Python Syntax

In this section we will begin to dive into Python syntax. I'll assume you have some questions.

What on Earth is syntax?

In programming, syntax refers to the rules that dictate the structure and organization of code. It determines how statements should be written to be valid and understandable by the programming language interpreter or compiler.

Understanding syntax is crucial because even minor errors in syntax can cause the program to behave unexpectedly or fail to run altogether. Common syntax errors include misspelling keywords, missing punctuation, and improper indentation.

By mastering syntax, you'll be able to write clean, readable code that is easier to maintain and debug.

Okay, so?

Now that you have a basic understanding of what syntax is, let's explore some fundamental elements of Python syntax.

  • Variables: Used to store data values that can be manipulated or referenced in the program.
  • Data types: Define the type of data that can be stored in variables, such as integers, floats, strings, and booleans.
  • Operators: Symbols or keywords used to perform operations on variables and values, such as arithmetic, comparison, and logical operations.

Throughout this course, we'll delve deeper into each of these concepts and learn how to use them effectively in Python programming.

Indentation

Indentation is an essential aspect of Python syntax. It is used to define the structure and hierarchy of code blocks, such as loops, conditionals, and function definitions.

What is indentation?

Indentation refers to the spaces or tabs at the beginning of a line of code that determine its level of nesting within a code block. In Python, indentation is used to signify the beginning and end of code blocks, eliminating the need for explicit delimiters like curly braces ({}) found in other programming languages.

Indentation is extremely important and often is the cause for many issues running code, by ensuring that we always practice proper indentation, we can prevent many mistakes that will save you from headaches later!

To the right of this. you will see a quick example of this in action!

Example:

Correct:

if 5 > 2:
    print("TRUE!")

Incorrect:

if 5 > 2:
print("TRUE!")

In the above example, the first statement is correct as it uses indentation properly before the print statement. The amount of spaces can differ and as the programmer you can decide, the common amount of spaces is four, and most programmers use this as the standard.
