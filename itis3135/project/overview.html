Python Overview

What is python?

Python is a programming language that is widely recognized and renoun for its simplicity and reliability. Don't let this fool you, while simplistic, Python is extremely functional. It is used in a variety of fields including web-development, data science/analysis, artificial intelligence, and so much more.

Why learn Python?

Python was designed to be quite readable and has a relatively user friendly syntax. It reads very similar to the english language. Because of this, it allows for more robust commands to be executed in fewer lines of code than other languages.

But first, what is an IDE?

An IDE, or Integrated Development Environment, is a software application that provides comprehensive facilities to programmers for software development.

It typically consists of a code editor, a compiler or interpreter, build automation tools, and debugger, all integrated into a single user interface.

Using an IDE offers several advantages:

  • Enhanced productivity due to features like syntax highlighting, code completion, and automated refactoring.
  • Built-in debugging tools that help identify and fix errors in the code.
  • Integration with version control systems for collaborative development.
  • Support for various programming languages and frameworks.

Popular IDEs for Python development include:

  • Visual Studio Code
  • PyCharm
  • Spyder

Ultimately, the choice of IDE depends on individual preferences and requirements. It's a good idea to explore different IDEs to find the one that best suits your workflow and coding style.

Lets break this down. It may look weird at first but we will take it step by step!

defthis is a keyword in python used to define a function.

hello_worldthis text here is used to define the function's name.

( ) :The parenthesis (empty in this example) are used to set parameters for the function, and the colon is to define where the function's executable code begins.

printthis is a built in Python function that is used to display information or content to the screen.

("Hello, World!")Hello, World! would be the exact text that is displayed to the screen on execution of the function.

Don't worry if some of this does not make sense to you right now, we will most certainly go into detail throughout this material to develop a solid understanding of all of these sections of code!
