Python Overview

What is python?

Python is a programming language that is widely recognized and renoun for its simplicity and reliability. Don't let this fool you, while simplistic, Python is extremely functional. It is used in a variety of fields including web-development, data science/analysis, artificial intelligence, and so much more.

Why learn Python?

Python was designed to be quite readable and has a relatively user friendly syntax. It reads very similar to the english language. Because of this, it allows for more robust commands to be executed in fewer lines of code than other languages.

Lets take a quick look at some Python!

def hello_world():
    print("Hello, World!")

Lets break this down. It may look weird at first but we will take it step by step!

defthis is a keyword in python used to define a function.

hello_worldthis text here is used to define the function's name.

( ) :The parenthesis (empty in this example) are used to set parameters for the function, and the colon is to define where the function's executable code begins.

printthis is a built in Python function that is used to display information or content to the screen.

("Hello, World!")Hello, World! would be the exact text that is displayed to the screen on execution of the function.

Don't worry if some of this does not make sense to you right now, we will most certainly go into detail throughout to develop a solid understanding of all of these sections of code!

Quick note: The colors on the text are for visual purposes, your programming application will most likely do this, but the colors will most certainly differ.

Remember:

The goal here is to get a basic overview and begin a programming journey, not become an expert immediately. You will experience ups and downs as does every programmer that has ever existed, and will exist. It's only when mistakes are made, that you really begin learning. There are many approaches to solving a problem and going through problems will show you them will really show you that there is no "one size fits all" answer.
