Project Overview

Project Overview:

  • 1. The purpose of this website is to teach a fellow non computer science student how to program. I chose Python as the language.
  • 2. The intended users of this website is anyone who would like a basic overview of programming using the Python language.
  • 3. For this website I plan to have multiple pages. Each page will have a multiude information. I also plan to figure out how to implement a little test page, in that a user will be asked basic questions and will be able to answer in an input box and be graded.
